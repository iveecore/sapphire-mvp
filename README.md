# Sapphire MVP - Women's Lifestyle OS

## Quick Start
```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Stack
- Next.js 16 + React 19 + TypeScript
- Supabase (PostgreSQL)
- Tailwind + ShadcnUI
- OpenRouter (AI)

## Features
- User auth (email/password)
- AI Stylist (questionnaire → recommendations)
- Marketplace (browse + shop)
- Community feed (see others' looks)
- Gamification (quests, achievements)
- Privacy-first (trust dashboard)
- Admin panel (manage inventory)
