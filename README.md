# Sapphire MVP

Sapphire is the first consumer application built on IVEE OS.

IVEE is the company and platform architecture. IVEE OS is the identity, permission, memory, context, trust, and decision layer. Sapphire is Product 001: a consumer experience for fashion, confidence, wardrobe, shopping, community, and everyday life decisions.

## Quick Start
```bash
npm install
npm run dev
```

Visit http://localhost:3000

## Stack
- Next.js 16 + React 19 + TypeScript
- Supabase (PostgreSQL)
- Tailwind + ShadcnUI
- OpenRouter (AI)

## Current MVP Surface
- Public landing page
- Signup and login pages
- Style quiz
- Dashboard
- Community page
- Initial Supabase schema
- Mock recommendation API
